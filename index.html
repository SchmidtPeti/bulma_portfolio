Képességeim Projektek Tanulmányok Hobbijaim Kapcsolat

Képességek

HTML/CSS

Ez volt az első lépésem a webfejlesztési világban, és azt mondhatom, hogy biztos alapokkal rendelkezem ezen a területen. Weboldalakat hozok létre, amelyek minden eszközön jól néznek ki és jól működnek.

Sass

A Sass használatával gyorsabban és hatékonyabban tudok stíluslapokat készíteni. A Sass segítségével a CSS kódokat jobban átláthatóvá és könnyebben karbantarthatóvá tehetem.

JavaScript

Az alap JavaScript kódolástól kezdve a komplex frontend feladatokig - szeretem a JavaScript kihívásait. Az ES6 szintaktikájával és a nyelvben rejlő lehetőségekkel jól ismerem magam.

ReactJS

Az elmúlt évben elkezdtem tanulmányozni a ReactJS-t, és már alkalmaztam néhány projektben. A tanulás folyamatban van, de már biztosan értek a komponens alapú fejlesztéshez és a React állapotkezeléséhez.

Git

A verziókövetés és a csapatmunka elengedhetetlen részének tartom a Git ismeretét. Kényelmesen használom a leggyakoribb Git parancsokat, és tudom, hogyan kell kezelni a merge conflictokat.

Responsive Design

Egy mai weboldalnak minden eszközön jól kell működnie. Én ezt a szemléletet alkalmazom minden projektben, amit végzek.

Projektek

1. Filmtár

@filmtar_projekt

Technológia: ReactJS

Egy online filmadatbázis, amit ReactJS-ben készítettem. Az alkalmazásban lehetőség van filmek keresésére, kategóriák szerinti szűrésre, és a felhasználók elmenthetik a kedvenc filmjeiket egy saját listában. Ez a projekt nagy mértékben megerősítette a ReactJS és az API-kezelési ismereteimet.

2. Blog platform

@blogplatform_projekt

Technológia: HTML/CSS, JavaScript

Egy egyszerű blog platformot hoztam létre, ahol a felhasználók létrehozhatnak saját bejegyzéseket és kommentálhatnak másokét. Ez a projekt megerősítette az alap HTML/CSS és JavaScript ismereteimet, valamint a CRUD (Create, Read, Update, Delete) műveletek gyakorlására is lehetőségem nyílt.
